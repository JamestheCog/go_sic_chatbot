# 🤖 Chatbot for Serious Illness Conversations

A web application centered around an empathetic assistant capable of conducting serious illness conversations with advanced cancer patients' caregivers via open-ended conversations and resources. 

## 👓 Overview

The application connects advanced cancers' caregivers with a Gemini-3-Flash assistant designed to help its target demographic to help navigate complex emotional challenges 
that arise as a result of caring for somebody with advanced cancer.  This application is also a safe haven for caregivers to ask questions, express concerns, and receive thoughtful guidance during difficult times.

## 💡 Features

1. **Specialized conversation support**: underlying assistant is engineered to be empathetic and patient when handling sensitive discussions about serious illness conversations in advanced cancers.
2. **Image and voice inputs**: users can interact with the bot via image uploads and voice inputs.
3. **Conversation fetching and history**: users are able to view and reset their previous conversations.
1/ **Night mode toggling**: users can interact with the application via light or night mode (depending on the time of day or user preferences).

## 🧑 Target Users

The caregivers of advanced cancers' patients in the SPRinT department.


# ▶️ Getting Started

Getting started is easy no thanks to Go's package system.  Assuming that you've cloned this repository onto your machine, run the following command 
in your command line after ensuring that you have Go installed on your machine (and your terminal's working directory has been set to this directory) - 
to download the project's dependencies and start the application:

```go
go mod tidy
go run main.go
```

And that's it!
